# Site Institucional do IFSP - Câmpus Pirituba

Projeto desenvolvido para as aulas de Web Client, criando um site institucional para o IFSP Câmpus Pirituba.

## 📋 Sobre o Projeto

Site institucional com informações sobre o IFSP Câmpus Pirituba, incluindo cursos oferecidos, eventos, biblioteca e informações de contato. O projeto foi desenvolvido em etapas, conforme os requisitos das aulas 03 e 04.

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3
- Favicon

## 🎯 Funcionalidades Implementadas

### Aula 03 - Primeiros Passos
- ✅ Utilização de imagens no site
- ✅ Favicon personalizado
- ✅ Estruturação com divs
- ✅ Parágrafos e textos informativos
- ✅ Organização básica do conteúdo

### Aula 04 - Estrutura Completa
- ✅ Imagens em todas as páginas
- ✅ Hiperlinks para navegação
- ✅ Conexão entre páginas HTML (Principal, Cursos e Sobre)
- ✅ Tags semânticas (header, nav, main, section, footer)
- ✅ Menu de navegação fixo no cabeçalho
- ✅ Menu superior direito reutilizado em todas as páginas
- ✅ Tabelas para organização de informações
- ✅ Áudio institucional
- ✅ Listas para apresentação de conteúdos
- ✅ Botão "Voltar ao Topo" com ícone SVG

## 🎨 Paleta de Cores

O projeto utiliza uma paleta de cores moderna e institucional:

- **Verde Principal:** `#2F9E41` - Cabeçalho, botões, títulos
- **Verde Claro:** `#D7E087` - Destaques e efeitos hover
- **Vermelho:** `#CD181D` - Efeitos hover e elementos de destaque

## 📄 Páginas do Site

### Página Principal (index.html)
- Banner institucional
- Cards com os 5 cursos oferecidos
- Tabela de eventos
- Seção da biblioteca com link para biblioteca digital
- Seção do SUAP com vídeo incorporado
- Navegação interna com links âncora

### Página de Cursos (cursos.html)
- Lista completa de cursos superiores
- Descrição detalhada de cada curso
- Tabela de horários das aulas
- Áudio com informações sobre os cursos

### Página Sobre (sobre.html)
- História do Câmpus Pirituba
- Missão, visão e valores
- Infraestrutura detalhada
- Números e estatísticas
- Informações de contato e localização
- Como chegar ao câmpus

## 🎨 Recursos Visuais

- **Cards de cursos:** Design limpo sem bordas verdes (conforme solicitado)
- **Botões interativos:** Efeitos hover com mudança de cor
- **Ícones SVG:** Instagram e botão voltar ao topo
- **Responsividade:** Adaptação para dispositivos móveis
- **Animações suaves:** Fade-in e transições

## 📱 Responsividade

O site é totalmente responsivo e se adapta a diferentes tamanhos de tela:
- **Desktop:** Layout completo com menu horizontal
- **Tablet:** Ajustes no espaçamento e menu
- **Mobile:** Menu em coluna e elementos redimensionados
